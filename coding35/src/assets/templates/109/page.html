The DMux4Way gate is a digital logic gate that takes one input and distributes it to one of four outputs based on the value of the select input. 

            |in | sel  | a | b | c | d |
            | 0 |  00  | 0 | 0 | 0 | 0 |
            | 0 |  01  | 0 | 0 | 0 | 0 |
            | 0 |  10  | 0 | 0 | 0 | 0 |
            | 0 |  11  | 0 | 0 | 0 | 0 |
            | 1 |  00  | 1 | 0 | 0 | 0 |
            | 1 |  01  | 0 | 1 | 0 | 0 |
            | 1 |  10  | 0 | 0 | 1 | 0 |
            | 1 |  11  | 0 | 0 | 0 | 1 |
            ...

HDL

        CHIP DMux4Way {
            IN in, sel[2];
            OUT a, b, c, d;
        
            PARTS:
                DMux(in=in, sel=sel[1], a=c1, b=c2);
                DMux(in=c1, sel=sel[0], a=a, b=b);
                DMux(in=c2, sel=sel[0], a=c, b=d);
        }
