A HalfAdder is a digital logic gate that adds two binary digits and produces a sum and carry output. The sum output

HDL

        CHIP HalfAdder {
            IN a, b;    // 1-bit inputs
            OUT sum,    // Right bit of a + b 
                carry;  // Left bit of a + b
        
            PARTS:
            XOr(a=a, b=b, out=sum);
            And(a=a, b=b, out=carry);
        }
