Invented by Sir Tony Hoare in 1965. The null reference also known as "The billion-dollar mistake" is a special value that can be assigned to any reference type and its usage indicates that the reference is not referring to any object.

The null reference is a source of many bugs and can be avoided by using the Null Object Pattern. The Null Object Pattern is a behavioral design pattern that defines a null object that can be used in place of a null reference.
