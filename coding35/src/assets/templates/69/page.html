HTTP Protocol overview.

HTTP/1.1

  • Single TCP connection for each request
  • 6-10 simultaneous connections per domain

HTTP/2

  • Always secure
  • Single TCP connection for all requests
  • 64% reduction in page load times

HTTP/3

  • UDP
  • Multiple streams
  • Longer active connections
  • Fast connection establishment
