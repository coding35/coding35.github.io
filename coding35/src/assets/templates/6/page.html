In object-oriented programming, object composition is a technique for creating complex objects by combining simpler objects or components. Unlike object inheritance where a class inherits the properties and behavior of a parent class, object composition creates the situation where a class contains a reference to one or more objects, which it uses to achieve its desired functionality. Sub-objects or components are combined to form the new object, and each provides a specific set of features or behaviors.

The benefit of object composition over inheritance is that it allows developers to create modular flexible code that can be reused and combined in different ways. If done correctly, it can also help to simplify the design and implementation of complex systems, as each component can be developed and tested independently, and then combined with other sub-objects or components to create an object with desired functionality.

Object composition encourages encapsulation, loose coupling, and keeps each component focused on a single task. Unlike inheritance, class hierarchies will remain lightweight and easy to maintain, as each class will only contain a reference to a sub-object or component, rather than inheriting all the properties and behaviors of the parent class that can become complex and difficult to manage over time.

In the following example, a car object is composed of two sub-objects or components, the engine and the brake assembly objects of which each in their own right could be composed of other sub-objects or components. When the car object is created, it is composed of the engine and brake assembly objects providing the car with the desired functionality. The car object can then use the engine and brake assembly objects to achieve the desired functionality.

Example

            #include <iostream>
            using namespace std;
            
            class Engine {
            public:
                void start() {
                    cout << "Engine started." << endl;
                }
                void stop() {
                    cout << "Engine stopped." << endl;
                }
                void throttle_up() {
                    cout << "Fuel injected." << endl;
                }
                void throttle_down() {
                    cout << "Fuel restricted." << endl;
                }
            };
            
            class BrakeAssembly {
            public:
                void apply() {
                    cout << "Brake applied." << endl;
                }
                void release() {
                    cout << "Brake released." << endl;
                }
            };
            
            class Car {
            private:
                Engine engine;
                BrakeAssembly brake;
            public:
                void start() {
                    engine.start();
                }
                void stop() {
                    engine.stop();
                }
                void accelerate() {
                    engine.throttle_up();
                }
                void decelerate() {
                    engine.throttle_down();
                }
                void depress_brake_peddle() {
                    brake.apply();
                }
                void release_brake_peddle() {
                    brake.release();
                }
            
            };
            
            int main() {
                Car car;
                car.depress_brake_peddle();
                car.start();
                car.release_brake_peddle();
                car.accelerate();
                car.decelerate();
                car.depress_brake_peddle();
                car.stop();
                return 0;
            }
