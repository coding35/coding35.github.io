XOR also known as the exclusive OR gate is a digital logic gate that outputs true only when the number of true inputs is odd.

HDL

            CHIP Xor {
              IN a, b;
              OUT out;
          
              PARTS:
                Or(a=a, b=b, out=outa);
                Nand(a=a, b=b, out=outb);
                And(a=outa, b=outb, out=out); 
          }
