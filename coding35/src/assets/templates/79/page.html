Comparison between Singleton and Static Classes

Singleton Static Class
Can implement interfaces No interfaces, just a collection of static methods and functions
Can be passed as a arguments to methods Cannot be passed as a arguments
Can be dependancy injected and used in other design patterns Cannot be injected
Can be assigned to variables Cannot be assigned to variables
Supports polymorphism Purely procedural
Can have state Can only access global state
Can be serilized which will allow them to be stored, transferred over the wire No support for serialization
