A register is a sequential device that stores a binary number. It is composed of multiple data flip-flops (DFF). The register HDL illustrates the construction of a 16-bit register using 16 single bit registers (Bit Register).

This example shows 16-bit Register truth table.

HDL

        CHIP Register {
            IN in[16], load;
            OUT out[16];
        
            PARTS:
                Bit(in=in[0], load=load, out=out[0]);
                Bit(in=in[1], load=load, out=out[1]);
                Bit(in=in[2], load=load, out=out[2]);
