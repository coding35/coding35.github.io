A basic Logic Gate that produces an output that is true only if all its inputs are true.

HDL

            
            CHIP And {
              IN a, b;
              OUT out;
              
              PARTS:
               Nand(a=a, b=b, out=c);
               Not(in=c, out=out);
          }
