The Adapter Pattern converts an interface of a class into the interface the client expects to allow incompatible code interfaces to work together. This is useful when you want to use an existing class and need to provide an interface the client is expecting. In addition to enabling code compatibility, adapters can also allow clients to make use of new libraries and subsets without changing any code.
