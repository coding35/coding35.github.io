A single bit register is a sequential device that stores a single bit of data. It is composed of a data flip-flop (DFF) and a multiplexer (Mux). The Mux is used to control the input of the DFF. When the load input is set to 1, the Mux will pass the input value to the DFF. When the load input is set to 0, the Mux will pass the output value of the DFF to itself. The output of the DFF is the output of the single bit register. 

        | time |in |load|out|
        | 0+   | 0 | 0  | 0 |
        | 1    | 0 | 0  | 0 |
        | 1+   | 0 | 1  | 0 |
        | 2    | 0 | 1  | 0 |
        | 2+   | 1 | 0  | 0 |
        | 3    | 1 | 0  | 0 |
        | 3+   | 1 | 1  | 0 |
        | 4    | 1 | 1  | 1 |
        | 4+   | 0 | 0  | 1 |
        | 5    | 0 | 0  | 1 |
        | 5+   | 1 | 0  | 1 |
        | 6    | 1 | 0  | 1 |
        | 6+   | 0 | 1  | 1 |
        | 7    | 0 | 1  | 0 |
        | 7+   | 1 | 1  | 0 |
        | 8    | 1 | 1  | 1 |
        | 8+   | 0 | 0  | 1 |

This example shows Single Bit Register truth table.

HDL

        CHIP Bit {
            IN in, load;
            OUT out;
        
            PARTS:
            Mux(a = t1 , b = in , sel = load , out = o);
            DFF(in = o, out = t1, out = out);
        }
