Pattern Description Classification Class/Object Pattern
Adapter Pattern Changes the interface of one or more classes. Structural Class
Class Adapter Pattern Uses multiple inheritance to adapt one interface to another. Structural Class
Command Pattern Decouple the sender of the request from the object that executes it. Behavioral Object
Composite Pattern Clients treat collections of objects and individual objects uniformly. Structural Object
Decorator Pattern Allows for the dynamic wrapping of objects in order to modify their. behavior Structural Object
Facade Pattern Simplifies the interface of a group of classes Structural Object
Factory Method Pattern Creates an instance of several derived classes. Creational Class
Iterator Pattern Provides a way to traverse a collection of objects without exposing the collection's implementation. Behavioral Object
Observer Pattern Allows a group of objects to be notified when some state changes. Behavioral Object
Proxy Pattern Wrap an object to control access to it. Structural Object
State Pattern Encapsulates state-based behavior and uses delegation behavior to the current state. Behavioral Object
Strategy Pattern Encapsulates interchangeable behaviors and uses delegation to decide. which one to use Behavioral Object
Template Method Pattern Create a template for an algorithm and let subclasses redefine certain steps. Behavioral Class

Classifications

  • Creational Patterns: Involves object instantiation and a way to decouple a client from the objects it needs to instantiate.
  • Behavioral Patterns: How classes and objects interact and distribute responsibility.
  • Structural Patterns: How classes and objects are composed to form larger structures.
  • Class Patterns: How relationships between classes are defined via inheritance. Relationships in class patterns are established at compile time.
  • Object Patterns: Describes relationships between objects and are primarily defined by composition. Relationships in object patterns are typically created at runtime and are more dynamic and flexible.
