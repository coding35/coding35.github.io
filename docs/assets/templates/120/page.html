A data flip flop or DFF is an elementary sequential device that is the basic building block component used to construct memory chips. The clock is used to pulse the DFF, which causes the DFF to store the input value. The DFF has two inputs, a data input and a clock input. Due to the speed of the signal from the clock, two inverters are used to delay the signal just long enough for the AND gate to produce a high output.
